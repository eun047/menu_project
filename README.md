# 🍽 메뉴 추천 앱

오늘 뭐 먹을지 고민될 때, 상황과 취향에 맞는 메뉴를 추천해주는 웹 앱입니다.

## 🔗 배포 링크
https://menu-project-1jdb.onrender.com/

## ✨ 기능
- 상황 기반 추천 : 인원 수와 시간대(아침/점심/저녁)을 입력하면 그에 적합한 메뉴를 추천해줍니다.
- 태그 기반 추천 : 면, 국물, 고기 등 원하는 특성을 선택하면 선택한 특성들 중에 하나라도 해당되는 메뉴를 추천해줍니다.

## 🛠 기술 스택
- Python / Flask
- HTML / CSS / JavaScript
